* 오토스케일링 비용 (https://aws.amazon.com/ko/autoscaling/pricing/)
AWS Auto Scaling에는 추가 비용이 없습니다. 
애플리케이션을 실행하는 데 필요한 AWS 리소스와 Amazon CloudWatch 모니터링 요금만 지불하면 됩니다.

* 오토스케일링 조정 정책
평균 CPU 사용률, 평균 네트워크 입·출력, 대상당 ALB 요청 수 + Custom Metric이 가능한 듯 + CW Agent(?)

* 기존에 쓰고있던 인스턴스 IP를 가지고 오토스케일링 하고 싶은 경우
인스턴스 우클릭 -> 인스턴스 설정 -> AutoScaling Group 연결 해서 새로운 그룹 생성 

* 인스턴스 중지 시 유의할 점
오토스케일링 그룹안에 있는 인스턴스를 중지할 경우 자동으로 확장되면서 '축소 보호'와 상관 없이 종료가 되기 때문에
인스턴스를 오토스케일링 그룹에서 '분리'하고 중지를 해야한다. (인스턴스 타입 확장 등을 위해 중지하는 경우가 있겠지)
